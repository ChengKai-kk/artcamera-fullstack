# 项目概览

## ArtCamera 是什么？

ArtCamera 是一个面向自助终端的 Web 应用，用户可拍照、选择风格，并生成 AI 艺术照片。系统包含 Vue 3 + Vite 前端与 Node.js Express 后端，后端负责对接多家图像生成服务。

## 核心目标

- 引导用户完成“选择风格 -> 拍摄 -> 生成 -> 保存”的简单流程
- 统一不同 AI 提供方的调用方式
- 支持大屏、自助终端场景（触屏、全屏、无操作超时）

## 技术栈

- 语言：JavaScript
- 前端：Vue 3、Vite、Vue Router
- 后端：Node.js 18+、Express
- 部署：GitHub Pages（前端）、阿里云函数计算（后端）

## 关键功能

- 大屏自助 UI 与轮播主视觉
- 摄像头拍摄、倒计时与重拍
- AI 生成任务创建与轮询
- 结果页二维码保存与无操作返回
- 多 Provider 接入（DashScope / Doubao / Banana）

## 项目结构

- artcamera_front/
  - src/（应用代码）
  - public/（静态资源与风格配置）
- artcamera_backend/
  - index.js（API 服务）

## 快速开始

前端（在 `artcamera_front/` 目录）：

- 安装依赖：`npm install`
- 启动开发：`npm run dev`
- 构建产物：`npm run build`
- 本地预览：`npm run preview`

后端（在 `artcamera_backend/` 目录）：

- 安装依赖：`npm install`
- 启动服务：`npm start`

## 架构小结

前端是单页应用，页面间状态通过 sessionStorage 传递。后端提供统一 API，负责创建任务并查询结果，底层 Provider 通过环境变量选择并封装成统一接口。
