# ArtCamera Fullstack

ArtCamera 是一个面向自助终端的 Web 应用，用户可拍照、选择风格，并生成 AI 艺术照片。
前端基于 Vue 3 + Vite，后端是 Node.js Express，负责对接多家图像生成服务。

## 项目结构

- `artcamera_front/` 前端（Vue 3 + Vite）
- `artcamera_backend/` 后端（Express API）

## 快速开始

前端（在 `artcamera_front/` 目录）：

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

后端（在 `artcamera_backend/` 目录）：

- `npm install`
- `npm start`
